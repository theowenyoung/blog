以下这几个别名和函数节省了我至少一吨的时间。

```bash
function gg() { git commit -m "$*" -a }
function gp() { git commit -a -m "$*" && git push }
function ga() { git add . }
alias ll="ls -GF -l"
alias la="ls -GF -la"
alias gpull="git pull"
alias gpush="git push"
alias gd="git diff"
alias gs="git status 2>/dev/null"
alias gv="git remote -v"
```
